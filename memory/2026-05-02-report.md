### 📄 重点论文 1：Synthetic Computers at Scale（arXiv:2604.28181）

**标题**：Synthetic Computers at Scale for Long-Horizon Productivity Simulation

**一句话**：造1000个"假电脑"，让AI agent在上面模拟一个月的真实工作，用来训练AI的长时间任务能力。

**核心方法**：
- 批量生成"合成计算机"：完整文件系统（文件夹层级、文档/表格/演示文稿）
- 两个 Agent 协作：一个设定需要一个月完成的多目标，另一个扮演用户在这台电脑上持续工作（超2000轮对话，平均8小时）
- 用这些模拟数据训练 Agent，在同类和跨类任务上都有显著提升

**为什么值得关注（对老大学习 AI 框架代码有帮助）**：
- 这是一种全新的 AI 训练数据生成范式——不再用人类真实数据，而是**创造模拟环境**
- "合成用户 + 合成世界"的思路，和咱们学习 Agent 设计时理解"环境模拟"直接相关
- 方法论层面涉及：多 Agent 协作（Multi-Agent）、强化学习（RL for Agent）、长程任务规划

**难度评估**：中等——概念容易理解，工程规模惊人，适合作为"AI 训练数据合成"概念的入门案例。

### 📄 重点论文 2：PhyCo - Learning Controllable Physical Priors for Generative Motion（arXiv:2604.28169）

**标题**：Learning Controllable Physical Priors for Generative Motion

**一句话**：让 AI 视频生成不只是好看，还要符合物理规律（摩擦力、回弹、材质反应）。

**核心方法**：
- 100K+ 仿真视频数据集：系统变化摩擦、弹性、变形、力等参数
- Physics-supervised Fine-tuning：用 ControlNet 把物理属性图（像素级对齐）注入预训练扩散模型
- VLM-guided Reward Optimization：微调视觉语言模型评估生成视频的物理合理性并提供梯度反馈

**为什么值得关注**：
- 视频生成正在从"看起来对"走向"物理上正确"
- 控制物理参数的思路——用可解释的物理属性控制生成过程——在 Agent 设计里也有参考价值
- 100K 仿真视频的概念和论文1的"合成数据"思路呼应，都是用规模化仿真数据喂模型

**难度评估**：中等偏高——涉及扩散模型、ControlNet、物理仿真，对视频生成领域不熟悉的话有些门槛。但核心洞察很清晰：生成质量→物理一致性。
